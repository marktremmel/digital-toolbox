üîÄ Conditionals (If/Else)

Conditionals allow your program to make decisions based on conditions.

Pseudocode:

IF age >= 18 THEN
  PRINT "Adult"
ELSE
  PRINT "Minor"
END IF

Python:

if age >= 18:
    print("Adult")
else:
    print("Minor")

JavaScript:

if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

üîÅ Loops

  PRINT i

  console.log(i);

üìã Arrays/Lists

Arrays store multiple values in a single variable.

Pseudocode:

fruits = ["apple", "banana", "cherry"]
PRINT fruits[0] // prints "apple"

Python:

fruits = ["apple", "banana", "cherry"]
print(fruits[0]) # prints "apple"

JavaScript:

let fruits = ["apple", "banana", "cherry"];
console.log(fruits[0]); // prints "apple"

üìñ Dictionaries/Objects

Dictionaries store data as key-value pairs.

Pseudocode:

student = {
  "name": "Anna",
  "age": 15,
  "grade": "10th"
}

Python:

student = {
    "name": "Anna",
    "age": 15,
    "grade": "10th"
}
print(student["name"]) # prints "Anna"

JavaScript:

let student = {
  name: "Anna",
  age: 15,
  grade: "10th"
};
console.log(student.name); // prints "Anna"
